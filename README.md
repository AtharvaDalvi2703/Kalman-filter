# Kalman-filter

---

## Week 1: Foundations of Machine Learning & Statistics

Week 1 focuses on core theoretical and practical aspects of machine learning.

### Topics Covered
- Multiple Linear Regression and OLS derivation
- Bias–variance decomposition
- Multicollinearity and leverage analysis
- Fairness and bias detection in salary prediction
- Model interpretability using SHAP
- Deep neural networks for handwritten digit classification (MNIST)

### Key Skills
- Statistical reasoning
- Responsible AI and fairness metrics
- PyTorch-based neural network implementation

---

## Week 2: Kalman Filters & Algorithmic Trading

Week 2 applies machine learning to financial markets using adaptive models.

### Highlights
- Feature engineering on real-world stock data (MSFT)
- Time-varying parameter estimation using Kalman Filters
- Integration of Kalman-filtered states with LSTM models
- Rule-based trading signal generation
- Backtesting and performance evaluation
- Comparison against buy-and-hold benchmarks

### Focus
- Non-stationary financial time series
- Causal signal generation
- Practical trading constraints and evaluation metrics

---

## Week 3: Reinforcement Learning & Deep Q-Learning

Week 3 introduces reinforcement learning and explores the limitations of tabular methods.

### Components
- Q-learning parameter analysis using FrozenLake
- Exploration–exploitation trade-offs
- Failure of tabular Q-learning in continuous state spaces
- Deep Q-Network (DQN) implementation for MountainCar
- Experience replay and target networks
- Learning under delayed rewards

### Emphasis
- Conceptual understanding of RL dynamics
- Stability and generalization using function approximation
